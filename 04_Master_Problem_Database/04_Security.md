# Purpose

This document contains validated security-related problems discovered during research.

Many AI-assisted projects work correctly but fail to follow security best practices. These issues often remain unnoticed until deployment, making them costly and risky to fix later.

This document focuses on identifying common security mistakes, understanding their root causes, and documenting proven solutions.
